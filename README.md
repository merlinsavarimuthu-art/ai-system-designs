# 🚀 AI Tokenization Demo (Python)

This project demonstrates a **simple AI system built using tokenization**.

## 🧠 What This Project Does
- Tokenizes text using CountVectorizer
- Converts text into numeric vectors
- Trains a Naive Bayes classifier
- Predicts sentiment (Positive / Negative)

## 🔧 Tech Stack
- Python
- scikit-learn
- GitHub Codespaces

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
